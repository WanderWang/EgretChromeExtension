EgretChromeExtension
====================

### 如何使用
* 下载并解压到任意文件夹
* 打开 Chrome，在右上角点击 选项-工具-扩展 ， 或者在浏览器地址栏中输入 ``` chrome://extensions/ ```
* 在Extension中勾选 ```开发者模式 ```
* 在Extension中选择  ```加载正在开发的扩展程序 ```，选择本文件夹 
* 右上角可以看到扩展已经成功安装
* 执行 egret startserver 命令
* 在打开的Chrome浏览器调试页中点击扩展，即会自动编译，编译成功后会自动刷新浏览器
